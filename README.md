# Transkaotisk Aktion Website

## Running locally

Requires Node.js installed

```sh
$ git clone https://github.com/transkaotisk-aktion/website.git
$ cd website
# No NPM install required!

# Build into ./dist/
$ npm run build

# Serve (does not watch for changes, must restart)
$ npm run serve

# Build and serve
$ npm run dev
```
